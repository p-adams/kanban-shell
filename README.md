# kanban-shell

### Getting Started

Requirements:

Deno `^1.15.3`

To run locally:

`$ deno install --allow-net --allow-read https://deno.land/std@0.113.0/http/file_server.ts`

Follow the installation steps provided.

Then, in the project directory, run:

`$ file_server .`

Visit `http://localhost:4507/` in the browser.
